# Superstore Sales Performance Dashboard (Power BI)



This project is an interactive Sales Performance Dashboard built using Power BI based on the popular Superstore dataset.

The dashboard provides deep insights into:

* Sales trends 
* Profitability 
* Customer behavior 
* Product performance 
* Regional distribution 

It is designed to help businesses make data-driven decisions.

---
 Objectives

* Analyze overall sales and profit performance
* Identify top-performing products and categories
* Understand regional sales distribution
* Track customer purchasing patterns
* Enable interactive data exploration using filters & drill features

---
Dashboard Features

1. Sales Overview

* Total Sales KPI
* Total Profit KPI
* Total Orders
* Profit Margin

---

2. Visualizations

Sales Analysis

* Sales by Category (Bar Chart)
* Sales Trend (Year → Month Drill Down)
* Region-wise Sales (Map / Bar Chart)

Product Insights

* Top 10 Products by Sales
* Product Performance Table (Sales, Profit, Quantity)
* Category → Sub-category → Product Drill-down

Customer Insights

* Top Customers by Sales
* Customer Table (Orders, Sales)
* Region-wise Customer Distribution

---

Advanced Features (Power BI)

✔ Drill Down (Year → Quarter → Month)
✔ Drill Through (Product Detail Page)
✔ Custom Tooltip Page (Mini Insights on Hover)
✔ Top N Filtering (Top 10 Products)
✔ Slicer Sync Across Pages
✔ Conditional Formatting (Tables)

---

Filters (Slicers)

* Year
* Region
* Category

These slicers allow dynamic filtering across all dashboard pages.

1. Download the `.pbix` file

3. Use slicers to filter data
4. Hover over charts for tooltip insights
